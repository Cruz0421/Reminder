# Reminder

## What is the Remider App
```
Reminder application for CMPS 3390 Application Development 
```
## Key Features
```
  -Sends notifications
  -Set timer by time length
  -Repeat
  -Checklist
  -Mark when tasks are done
  -Remind again later
  -Categorical sorting
  -Sort reminders/checklists by group
  -Communicates with database 
  -User sign in
  -Data backup
  -Hosted on Odin
```
